20a. Error Code Card
FLT-4012 Delta Table Schema Mismatch Fatal
The schema of the source DataFrame does not match the target Delta table. Column "revenue_amount" expected type DECIMAL(18,2) but received DOUBLE. This error occurs during the write phase when schema enforcement is enabled.
Fix: Add .option("mergeSchema", "true") to your write operation, or cast the column before writing.
FLT-2001 Token Refresh Timeout Retriable
MWC token refresh exceeded the 30-second timeout. The service will retry automatically in 5 seconds. If this persists, verify your AAD configuration and network connectivity.
Fix: Check edog-config.json for correct tenantId and clientId values. Run edog token refresh to force a new token.
