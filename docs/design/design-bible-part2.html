Code & JSON

Dark code blocks with syntax highlighting, interactive JSON tree, and side-by-side diff view.

10a — Code block with syntax coloring and copy button
SQL 
SELECT
  t.table_name,
  t.row_count,
  ROUND(t.size_bytes / 1073741824, 2) AS size_gb,
  t.last_modified
FROM fabric_catalog.tables t
WHERE t.schema_name = 'default'
  AND t.row_count > 0
ORDER BY t.size_bytes DESC
LIMIT 20;

Empty States

Four distinct empty state patterns for no-data, disconnected, error, and first-run.

12a — No data
No workspaces found
You don't have access to any Fabric workspaces, or none match your filter.
12b — Disconnected
Deploy to enable this feature
DAG Studio requires a running FabricLiveTable service. Deploy to a lakehouse first.
12c — Error state
Could not load tables
The Fabric API returned an error while fetching table metadata.
HTTP 503 · Service temporarily unavailable
12d — First run
Get started by selecting a workspace
Choose a workspace from the explorer on the left to browse lakehouses and tables.
