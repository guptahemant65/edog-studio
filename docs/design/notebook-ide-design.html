12 Keyboard Shortcuts Reference

All keyboard interactions for the notebook IDE. Designed for keyboard-first workflow — no action requires a mouse.
Ctrl+EnterRun current cell
Shift+EnterRun cell, move to next
Ctrl+Shift+EnterRun all cells
Ctrl+SSave notebook
Ctrl+↑Move cell up
Ctrl+↓Move cell down
EscapeExit cell editing
EnterEnter cell editing
TabInsert 4 spaces (in cell)
Ctrl+Shift+MToggle markdown / code
Ctrl+DeleteDelete cell
Ctrl+KCommand palette
